- Instalar o plugin do maven no eclipse
  Help -> Eclipse Marketplace. Digite no campo search: "maven wtp" e instale o primeiro plugin

- Faça o clone do repositório* na pasta do workspace do seu eclipse.

- Importe o projeto no seu workspace
  File -> Import -> Maven -> Existing Maven Projects -> Selecione o seu projeto

- Instale o tomcat
  Aperte Ctrl + 3. Digite server, selecione essa opção; Adicione um novo tomcat e adicione a sua aplicação e incie o tomcat.
  
- Acesse a sua aplicação. http://locahost:8080/spring-maven-project e http://locahost:8080/spring-maven-project/welcome
